# Keerthi Chakri — Portfolio

**Live:** https://chakri-portfolio-ivory.vercel.app/

Backend and AI Engineer. Built with React, Vite, Tailwind CSS, and Framer Motion.

---

## Stack

- **React + Vite** — frontend framework
- **Tailwind CSS** — styling
- **Framer Motion** — scroll animations and transitions
- **EmailJS** — contact form (no backend needed)
- **Vercel** — deployment

---

## Sections

- **Hero** — intro with rotating title and animated character
- **About** — background, experience stats, focus areas
- **Experience** — scroll-animated timeline (desktop + mobile)
- **Projects** — LLM Gateway, Credit Risk ML
- **Skills** — tech stack
- **Contact** — EmailJS-powered form

---

## Local Setup

```bash
# Install dependencies
npm install

# Create .env file
cp .env.example .env
# Add your EmailJS credentials to .env

# Run dev server
npm run dev

# Build for production
npm run build
```

---

## Environment Variables

Create a `.env` file in the root:

```
VITE_SERVICE_ID=your_emailjs_service_id
VITE_TEMPLATE_ID=your_emailjs_template_id
VITE_PUBLIC_KEY=your_emailjs_public_key
```

Get these from [emailjs.com](https://emailjs.com) → Email Services + Email Templates.
